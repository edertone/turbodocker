# General purpose docker images

This repository contains different general purpose docker images

Check readme file inside each docker image for more information on usage

## Images

- [file-tools](./file-tools/README.md) - General purpose utilities to work with several file formats like PDF, ZIP, etc..

- [php7-apache-vhosts](./php7-apache-vhosts/README.md) - Apache production ready server with multi host cappabilities, so we can host multiple websites on the same docker image

- [translation-service](./translation-service/README.md) - General purpose translation microservice to centralize application translations
